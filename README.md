# Yummy

A new Flutter project.

## 📱 Features
* **Custom Theming**: Implementation of a unified `FooderlichTheme` for dark and light modes.
* **Complex Layouts**: Usage of `Stack`, `Positioned`, `Column`, and `Row` to create layered UI elements.
* **Modular UI**: Organization of code into reusable custom widgets (`Card1`, `Card2`, and `Card3`).
* **Asset Management**: Proper integration and display of local images and custom fonts.

## 🛠 Tech Stack
* **Framework**: Flutter
* **Language**: Dart
* **Key Widgets used**:
    * `Scaffold` & `AppBar` for the app structure.
    * `Stack` for overlaying text on images.
    * `Container` with `BoxDecoration` for styling and borders.
    * `CircleAvatar` for user profile visuals.

## 🚀 How to Run
1.  Clone the repository (if you haven't already):
    ```bash
    git clone [https://github.com/aruzhanimm/recipe_calculator.git](https://github.com/aruzhanimm/recipe_calculator.git)
    ```
2.  Switch to the Chapter 3 branch or folder.
3.  Install dependencies:
    ```bash
    flutter pub get
    ```
4.  Launch the app:
    ```bash
    flutter run
    ```
